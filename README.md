# Style Matrix - Salon Management System

A comprehensive salon management system built with HTML, CSS, JavaScript, and Supabase.

## Setup Instructions

1. Copy `.env.example` to `.env` and fill in your Supabase credentials
2. Run the SQL schema in your Supabase SQL editor
3. Configure RLS policies as specified in the documentation
4. Deploy to Netlify

## Features

- Admin dashboard with real-time sales tracking
- Worker management
- Service/costing management
- Calendar views with daily and monthly reports
- Transaction entry system
- Timezone support (GST/UTC+4)
